# FlutterWechat

Use Flutter to copy Wechat.

## Getting Started

- [Config Flutter](https://github.com/flutter/flutter)
- Connect your device
- Clone project `git clone https://github.com/lujiahao0708/FlutterWechat.git`
- Enter project folder `cd FlutterWechat`
- Run `flutter run`
